Steps to run this project file
1. Drag the Iris_Project file found in the zip file to your Matlab Folder
2. Opeb Matlab 
3. On the Current Folder Window right click the Iris_Project file and go to add path and select the option selected folder and subfolders
4. Open main.m in the Iris_Project file and run it which takes approximately 5 to 10 minutes as it creating the database

Critical Note: This project must be run in a 2015b Matlab Enviroment
